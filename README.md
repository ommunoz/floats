# Floats: Community Tab Protocol

Floats is a DeFi protocol on the Flow blockchain that lets patrons fund a "tab"
at local businesses which neighbors can draw from using a virtual card tap —
with no app install, no wallet, and no friction at the register.

_Built for the Flow DeFi Hackathon._

## Local Development

1. Start the Flow emulator: `flow emulator`
2. Deploy the local contracts: `flow project deploy --network emulator`
