# Floats Frontend

This is the Vue 3 + Vite frontend for Floats, built specifically for the Flow DeFi Hackathon demo.

## Tech Stack
- **Framework**: Vue 3 (Composition API)
- **Tooling**: Vite
- **Styling**: Pure SCSS
- **Web3 Integration**: `@onflow/fcl`

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

*Note: Ensure the local Flow emulator is running (`flow emulator`) and the backend services are up before testing FCL transactions.*
